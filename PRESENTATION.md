# Happy Coloring AI

### An AI-Integrated Paint-by-Numbers E-Commerce Platform

**Final Year Project — Information Technology, 2025–2026**

---

## Project Overview

Happy Coloring AI is a full-stack e-commerce web application that combines an online store with an AI-powered image generator. Users can browse and purchase ready-made digital paint-by-numbers kits, or describe their own ideas in text and have the AI generate a custom numbered coloring template instantly. The platform targets the growing global paint-by-numbers market (projected USD 1.2 billion by 2028) and removes the traditional barrier of being limited to pre-designed templates.

---

## Project Objectives

- Research and design the system architecture, database schema, and software requirements
- Build a fully functional online store with product browsing, shopping cart, checkout, and order management
- Integrate Google Gemini AI to generate custom paint-by-numbers artwork from user text descriptions
- Develop a comprehensive admin dashboard for managing products, orders, and users
- Implement secure user authentication and role-based access control
- Deploy the complete system to a cloud environment (Vercel + Render)

---

## Key Features

- AI Image Generation — Enter a text description; the system automatically generates a black-and-white numbered coloring template with a matching color palette
- Product Gallery — Browse a curated library of ready-made paint-by-numbers products across various themes
- Shopping Cart & Checkout — Full cart management with Cash on Delivery payment support
- Favourites — Save and revisit favourite products at any time
- Order Management — Real-time order tracking for both customers and admins
- AI Chatbot — Automated customer support assistant powered by AI
- Admin Panel — Full CRUD management for products, users, and orders with a statistics dashboard
- Secure Authentication — Email/password registration and login via Firebase Auth

---

## Technologies

- **Frontend:** Next.js 14, TypeScript, Tailwind CSS, Zustand
- **Backend:** Node.js, Express.js
- **Database & Auth:** Firebase Firestore, Firebase Authentication, Firebase Storage
- **AI:** Google Gemini API
- **Deployment:** Vercel (frontend), Render (backend)
